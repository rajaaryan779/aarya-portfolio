# Aarya Vaidya — Portfolio

Next.js portfolio with Framer Motion animations and Tailwind CSS.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
npm start
```

## Tech Stack
- **Next.js 14** (App Router)
- **Tailwind CSS**
- **Framer Motion** (animations)
- **react-intersection-observer** (scroll triggers)

## Sections
1. **Hero** — Typing animation, floating orbs, profile card
2. **About** — Timeline, stats, personal bio
3. **Projects** — SentimentIQ, CarbonTrack, VisionNet, FlowMind
4. **Skills** — Animated skill bars, tech badge cloud
5. **Contact** — Direct links + contact form (opens mailto)
